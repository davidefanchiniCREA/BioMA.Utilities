# OrderedSetOnceFixedKeyEnumerable&lt;TKey,TValue&gt; class

Represents a dictionary in which the keys are fixed at instantiation time, and for which, once a value has been configured for a key, it is not possibile to overwrite it. Guarantees iteration of the keys in the order they are configured in the constructor
